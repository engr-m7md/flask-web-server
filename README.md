\#Flask web server
helps you to download your local files from your computer to any device that connected to local network



\#Notes
make sure to copy all your files (that you want to transfer them to other device) to files Folder



\#steps

1. run the flaskMain.py files by double click on it or by the following command
   python flaskMain.py
2. will show cmd window (in windows OS) or termainal (in linux OS) make sure to copy the IP and Port on it to your browser on any device over network
3. you can browse your files in browser then click on any one to download it
